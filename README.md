- kubectl
- Helm
- linkerd
- boundary
- jq
- git
- bind-tools (nslookup et dig)
- vim

Cette image docker permet d'assister au hand's on [Le petit Chaperon rouge se met enfin au Zero Trust](https://github.com/Sphinxgaia/little-red-riding-hood/tree/lab-devoxx)
